## 🚀 How to Run

1. **Clone or Download** the project files
2. **Open** `index.html` in your web browser
3. **Alternative**: Use a local development server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

## 📱 Responsive Design

The website is fully responsive and optimized for:
- Desktop computers (1200px+)
- Tablets (768px - 1199px)
- Mobile devices (320px - 767px)

## 🎨 Customization

### Adding Your Photo
Replace `images/myphoto.jpg` with your professional photo

### Updating Social Links
Edit the social media URLs in the header and contact sections

### Adding Projects
1. Add project images to `images/portfolio/`
2. Update the portfolio section in `index.html`
3. Include project descriptions and links

### Download CV
Place your CV file in the `files/` directory and update the download link
