Selected Works

Here are some of my selected works I have done lately. Feel free to check them out.

FindMe

Mising person finding web application

FindMe, a comprehensive missing person finding platform consisting of a web application for police to register missing person cases and a mobile app for public participation. The web portal enables law enforcement to create detailed missing person profiles with photos and case information, while the mobile application allows citizens to upload photos of suspicious or potentially matching individuals. The system leverages community participation and technology to create a collaborative network for locating missing persons more efficiently.

ByteBooks

MERN full stack project

Byte Books, a comprehensive library management system for a college using the MERN stack (MongoDB, Express.js, React, Node.js). The application streamlines library operations by enabling students and staff to search, reserve, and manage book rentals efficiently. The system features user authentication, real-time book availability tracking, automated due date notifications, and administrative controls for librarians to manage inventory and generate reports.

The DROP

Expense tracking system

Drop, a custom expense tracking application for a clothing retail business, designed to simplify financial management for small business owners. The application enables automatic expense categorization, generates detailed spending reports, and provides real-time insights into cash flow patterns. Features include receipt scanning, budget alerts, and comprehensive analytics dashboards that help business owners make informed financial decisions and optimize their operational costs.

Delivery App UI

Flutter Mobile Application

A comprehensive Flutter-based food ordering application featuring a modern, intuitive interface for restaurant customers. The app includes a clean bottom navigation system with Home, Menu, Favorites, and Orders sections, supporting multiple ordering modes including delivery, pickup, and table dining experiences. Key features include an animated splash screen with authentication flow, a dynamic home dashboard with promotional carousels and quick actions, a comprehensive menu system with category-based organization and real-time search, detailed food item overlays with customization options and AR integration, a sophisticated cart and checkout experience with payment integration, QR code-based table dining functionality, and comprehensive order management with tracking capabilities. The app utilizes a consistent design system with Google Fonts Poppins typography, an orange (#FF6B00) primary color scheme, and includes smart meal recommendations, floating cart functionality, and multi-modal support for different service types.

Leaf & Lumina Restaurant Website UI

Figma Web Design

A modern and responsive restaurant website concept designed in Figma, focused on delivering an elegant and user-centered digital dining experience. The design emphasizes visual storytelling through rich food imagery, balanced typography, and clean, structured layouts that capture the brand's warmth and sophistication. It features sections such as Home, About, Menu, Gallery, Reservations, and Contact, each crafted for intuitive navigation and visual consistency. The homepage highlights the restaurant's identity with a bold hero banner, featured dishes, and clear call-to-action buttons, while the menu section follows a category-based structure with detailed item views, subtle hover animations, and an engaging browsing flow. The ordering interface visually represents an easy and appealing process for selecting meals, viewing the cart, and choosing delivery or pickup. Using Poppins typography, a warm orange (#FF6B00) accent color, and soft neutral tones, the design maintains harmony with the brand's aesthetic and aligns seamlessly with the companion mobile app. Every element—from layout spacing to visual rhythm—was thoughtfully designed to provide a refined, modern, and enjoyable experience across devices.

iFashion E-commerce Website UI

Figma E-commerce Design

A contemporary and visually immersive eCommerce website design crafted in Figma, showcasing a refined shopping experience that blends elegance with functionality.
